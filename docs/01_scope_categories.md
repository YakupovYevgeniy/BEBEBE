## A) Categories & Scope

### Proposed Category List (v1)

| # | Category | Description | MVP? | Later? |
|---|----------|-------------|------|--------|
| 1 | **Health** | Track sleep, steps, mood, water intake, basic metrics | ✓ MVP | Advanced analytics, integrations |
| 2 | **Finance** | Budget overview, accounts summary, spending trends | ✓ MVP | Bank sync, recurring transactions |
| 3 | **Habits** | Daily habits, streaks, check-ins | ✓ MVP | Custom habit builder |
| 4 | **Tasks** | To-dos, priorities, due dates | ✓ MVP | Projects, subtasks |
| 5 | **Notes** | Quick notes, journal entries | ✓ MVP | Rich text, attachments |
| 6 | **Goals** | Long-term goals with milestones | Later | — |
| 7 | **Learning** | Track courses, books, skills | Later | — |
| 8 | **Relationships** | Contact reminders, important dates | Later | — |

### MVP Scope (What’s IN)

- **5 categories:** Health, Finance, Habits, Tasks, Notes
- Each category: list view, detail view, add/edit entry
- Home/dashboard with category cards and summaries
- Theme support (Day/Night)
- Local persistence only

### Out of Scope (MVP)

- AI/assistant features
- Cloud sync
- Third-party integrations (bank, wearables, etc.)
- Widgets
- iPad layout
- Watch app

### Later (Post-MVP)

- Goals, Learning, Relationships categories
- iCloud/CloudKit sync
- Bank API / HealthKit integration
- Home screen widgets
- iPad and watch support
