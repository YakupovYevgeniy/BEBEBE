## What is LIFE OS?

A modular Life Operating System iOS app — a central dashboard for managing key life domains: health, finance, habits, tasks, and notes.

## MVP Categories

1. **Health** — Sleep, steps, mood, water intake  
2. **Finance** — Account summaries, balances  
3. **Habits** — Daily habits, streaks, check-ins  
4. **Tasks** — To-dos, priorities, due dates  
5. **Notes** — Quick notes, journal entries  

## Tech Stack

- iOS: Swift + SwiftUI  
- Architecture: MVVM + Coordinators  
- Persistence: TBD (Stage 4)
